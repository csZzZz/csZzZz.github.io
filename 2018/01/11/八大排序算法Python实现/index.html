1.冒泡排序(BubbleSort)

  • 基本思想

    • 比较两个数的大小,较大的数下沉,较小的数冒起来.

  • 过程

    • 比较相邻的两个数据,如果第二个数小,就交换位置.
    • 从后向前两两比较,一直到比较最前两个数据.最终最小数被交换到起始的位置,这样第一个最小数的位置就排好了
    • 继续重复上述过程,一次将第2,3…n-1个最小数排好位置

  • 平均时间复杂度:O(n^2)

2.选择排序(SelectionSort)

  • 介绍
    • 选择排序无疑是最简单直观的排序,从待排序的数据元素中选出最小(或最大)的一个元素,放在序列的起始位置,直到全部待排序的数据元素排完
  • 步骤
    • 从未排序序列中找到最小(大)元素,存放到排序序列的起始位置.
    • 再从剩余未排序元素继续寻找最小(大)元素,然后放到已排序序列的末尾
    • 依次类推,直到所有元素均排序完毕
  • 平均时间复杂度O(n^2)
  • 代码实现
    @staticmethod
def selection_sort(input_list):
    length = len(input_list)
    for i in range(length):
        min = i  # 记录最小元素的下标
        for j in range(i + 1, length):
            if input_list[j] < input_list[i]:
                # 交换两个数
                input_list[i], input_list[j] = input_list[j], input_list[i]
    return input_list

3.快速排序(Quicksort)

基本思想:(分治)
  • 先从数列中取出一个数作为key值
  • 将比这个数小的数全部放在他的左边,大于或等于他的数全部放在他的右边
  • 对总有两个小数列重复第二部,直至各区间只有1个数
