1.Python的函数参数传递

看两个例子:

所有的变量都可以理解是内存中一个对象的”引用”.
通过 id来看 a的内存地址可以比较理解:

可以看到,在执行完 a = 2之后, a引用中保存的值,即内存地址发生变化,由原来 1 对象所在的地址变成了 2 这个实体对象所在的内存地址.

而第二个例子 a 引用保存的内存值就不会发生变化

这里的类型是属于对象的,而不是变量.而对象有两种, 可更改(mutable) 与 不可更改(immutable) 对象.
在python中, strings, tuples, numbers是不可更改的对象,
而 list, dict, set等则是可以修改的对象.

当一个引用传递给函数的时候,函数自动复制一份引用,这个函数里的引用和外边的引用没有半毛钱的关系.
所以第一个例子把引用指向了一个不可变的对象,当函数返回的时候,外面的引用没有感觉.
第二个例子函数内的引用指向的是可变对象,对它的操作就和定位了指针地址一样,在内存里进行修改.
